# Basic programming exercises using TypeScript

Excercises in this repository can help you understand basics of programming.
Files are organized according to their topics, e.g loops/for-loop/.
For each topic you will find simple example that will help you get started with tasks.

## Running scripts

In order to run scripts you have to use TypeScript compiler and Node.js. When you will have them installed compile your TypeScript file using `tsc` and then run js with `node`:
```sh
tsc.cmd ./file-name.ts
node ./file-name.ts
```
